# Contributing to QuickTask

Thank you for your interest in contributing to QuickTask! This document provides guidelines and instructions for contributing.

## ğŸš€ Getting Started

1. **Fork the repository**
2. **Clone your fork:**
   ```bash
   git clone https://github.com/yourusername/QuickTask.git
   cd QuickTask
   ```

3. **Create a branch:**
   ```bash
   git checkout -b feature/your-feature-name
   ```

4. **Make your changes**

5. **Test your changes:**
   - Test backend: `cd backend && npm test`
   - Test frontend: `cd frontend && npm test`
   - Test analytics: Run the service and verify endpoints

6. **Commit your changes:**
   ```bash
   git commit -m "Add: Description of your changes"
   ```

7. **Push to your fork:**
   ```bash
   git push origin feature/your-feature-name
   ```

8. **Open a Pull Request**
